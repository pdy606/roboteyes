# 🤖 똑똑한 서빙 로봇 점주 가이드 (v2.0)

사장님, 스마트 매장을 위한 로봇 뇌(AI) 관리가 이제 훨씬 쉬워졌습니다!
YOLOv8 인공지능이 매장의 메뉴인 **제육볶음, 돈까스, 김치찌개, 된장찌개**를 완벽하게 학습할 수 있도록 도와드립니다.

## 1. 학습 데이터 수집하기
포스기 화면의 **🧬 AI 학습 데이터 강화** 섹션을 확인해 주세요.
- 각 메뉴 이름 옆의 '파일 선택' 버튼을 눌러 실제 조리된 음식 사진을 업로드합니다.
- 사진이 많을수록(메뉴당 10~20장 이상 권장) 로봇의 인식 정확도가 올라갑니다.

## 2. 인공지능 재학습 (Training)
데이터를 올린 후 하단의 **⚡ 재학습 시작** 버튼을 누릅니다.
- **백그라운드 학습**: 학습 중에도 손님들의 주문을 계속 받을 수 있습니다. (무중단 서비스)
- **자동 교체**: 학습이 완료되면 시스템이 자동으로 새 모델을 적용합니다. 서버를 껐다 켤 필요가 없습니다!

## 3. 로봇 배차 시뮬레이션
로봇이 주방에서 음식을 인식하여 배차하는 과정을 테스트할 수 있습니다.
- **로봇 눈 (AI Vision)** 섹션에서 음식 사진을 업로드합니다.
- 로봇이 해당 메뉴를 가장 먼저 주문한 테이블로 자동 배차(Drive_to_Table_X)하는지 확인하세요.

---
> [!TIP]
> **정확도가 낮다면?**
> 그릇 무늬가 잘 보이게 찍거나, 조명이 밝은 곳에서 찍은 사진을 더 많이 추가한 뒤 재학습을 진행해 보세요!

> [!IMPORTANT]
> **시스템 재시작 시 유의사항**
> 컨테이너가 완전히 새로 생성(rm/run)될 경우, 수동으로 데이터를 다시 복사하거나 볼륨 마운트 설정을 확인해야 할 수 있습니다. 현재는 무중단으로 안전하게 관리되고 있습니다.
